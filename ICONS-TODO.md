# 图标文件待创建

需要创建以下图标文件：

1. **icon-192.png** (192x192px)
   - 用于 PWA 应用图标
   - 建议：试管图案，紫色渐变背景

2. **icon-512.png** (512x512px)
   - 用于 PWA 应用图标（高清）
   - 与 192px 版本设计一致

## 创建方法

### 方法 1：使用在线工具
访问 https://www.favicon-generator.org/ 或 https://realfavicongenerator.net/

### 方法 2：使用设计软件
使用 Figma、Photoshop 或其他设计工具创建

### 方法 3：使用 Emoji
将游戏中的试管 emoji 🧪 转换为图标：
- 访问 https://emoji.gg/
- 搜索 test tube
- 下载并调整大小

## 临时方案

在图标创建之前，可以：
1. 注释掉 manifest.json 中的 icons 部分
2. 或使用简单的纯色图标占位

创建完成后替换即可。
