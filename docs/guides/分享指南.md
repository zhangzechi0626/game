## 分享方式二：生成二维码

二维码可以让朋友用手机扫一扫就能玩，非常方便！

### 为什么使用二维码？

- ✅ 手机扫描更方便，不用手动输入
- ✅ 可以打印出来，贴在教室或办公室
- ✅ 朋友聚会时可以展示在屏幕上，大家扫码一起玩

### 推荐的二维码生成工具

1. **草料二维码**（推荐）
   - 网址：https://cli.im/
   - 特点：免费、无广告、支持自定义样式

2. **联图网**
   - 网址：https://www.liantu.com/
   - 特点：简单快速，无需注册

3. **二维工坊**
   - 网址：https://www.2weima.com/
   - 特点：支持美化二维码，添加 logo

### 操作步骤（以草料二维码为例）

1. 访问 https://cli.im/

2. 选择 **"网址"** 类型

3. 在输入框中粘贴你的游戏网址：
   ```
   https://你的用户名.github.io/water-sort-puzzle/
   ```

4. 点击 **"生成二维码"**

5. 下载二维码图片：
   - 右键点击二维码
   - 选择 "图片另存为"
   - 保存到电脑

6. 分享二维码：
   - 发送到微信群
   - 打印出来
   - 设置为头像或背景

### 二维码使用场景

📱 **线上场景**：
- 发到微信群、朋友圈
- 发到 QQ 群、QQ 空间
- 发到微博、小红书

🖨️ **线下场景**：
- 打印贴在教室、办公室
- 做成海报或宣传单
- 在聚会时展示在投影仪上

## 分享方式三：短链接服务

GitHub Pages 的网址比较长，使用短链接可以让分享更简洁。

### 为什么需要短链接？

- 原始链接：`https://zhangsan.github.io/water-sort-puzzle/`（较长）
- 短链接：`https://d.cn/abc123`（简短易记）

### 推荐的短链接服务

1. **百度短网址**
   - 网址：https://dwz.cn/
   - 特点：国内访问快，永久有效

2. **新浪短网址**
   - 网址：https://sina.lt/
   - 特点：知名度高，稳定

3. **缩我短链接**
   - 网址：https://suowo.cn/
   - 特点：免费，支持自定义后缀

### 生成短链接步骤

1. 访问短链接服务网站（如 https://dwz.cn/）

2. 在输入框中粘贴你的游戏网址

3. 点击 **"生成短链接"** 或 **"缩短"**

4. 复制生成的短链接

5. 测试短链接是否正常跳转

⚠️ **注意事项**：
- 短链接可能会被微信拦截，建议测试后再大规模分享
- 保存好短链接，避免忘记
- 部分短链接服务可能要求注册

## 添加到手机主屏幕

让游戏像 APP 一样，放在手机桌面上！

### iOS 设备（iPhone/iPad）

1. 用 Safari 浏览器打开游戏网址

2. 点击底部的 **"分享"** 按钮（一个向上的箭头）

3. 向下滚动，找到 **"添加到主屏幕"**

4. 编辑名称（如 "水排序"）

5. 点击右上角的 **"添加"**

6. 游戏图标会出现在桌面上，点击即可直接玩

✨ **优点**：
- 像原生 APP 一样打开
- 没有浏览器地址栏，全屏体验
- 加载速度更快（有缓存）

### Android 设备

1. 用 Chrome 浏览器打开游戏网址

2. 点击右上角的 **"⋮"** （三个点）

3. 选择 **"添加到主屏幕"**

4. 编辑名称

5. 点击 **"添加"**

6. 游戏图标会出现在桌面上

💡 **其他浏览器**：
- **华为浏览器**：菜单 → 添加至桌面
- **小米浏览器**：菜单 → 添加书签 → 添加到桌面
- **OPPO/VIVO 浏览器**：菜单 → 添加到桌面

### PWA 功能说明

如果你上传了 `manifest.json` 文件，游戏支持 PWA（渐进式 Web 应用）功能：

- 🚀 离线访问（玩过一次后，没网也能玩）
- 📲 桌面图标（使用自定义图标）
- 🔔 推送通知（可选，需要额外配置）
- ⚡ 快速启动（像原生 APP 一样）

## 常见问题

### Q1: 朋友打开链接显示 404？

**可能原因**：
1. GitHub Pages 还没生成完毕
2. 朋友的网络无法访问 GitHub
3. 链接复制错误

**解决方法**：
- 确认链接格式正确：`https://用户名.github.io/仓库名/`
- 自己先测试一下链接是否能打开
- 如果朋友网络问题，建议生成二维码或短链接
- 等待 3-5 分钟后再尝试

### Q2: 手机上显示不正常？

**可能原因**：
1. 手机浏览器兼容性问题
2. 屏幕尺寸适配问题
3. 背景图片加载慢

**解决方法**：
- 建议使用 Chrome、Safari、微信内置浏览器
- 检查 index.html 中的 viewport 设置
- 压缩背景图片，提高加载速度
- 清除浏览器缓存后重试

### Q3: 可以自定义域名吗？

可以！使用自己的域名会更专业。

**步骤**：
1. 购买域名（如 `game.yourdomain.com`）
2. 在仓库根目录创建 `CNAME` 文件
3. 文件内容填写你的域名（不带 https://）
4. 在域名管理处添加 CNAME 记录，指向 `你的用户名.github.io`
5. 等待 DNS 解析生效（可能需要 10 分钟到 24 小时）

**示例**：
```
CNAME 文件内容：
game.mywebsite.com

域名解析设置：
类型: CNAME
主机记录: game
记录值: zhangsan.github.io
```

### Q4: 如何查看访问量？

GitHub Pages 默认不提供访问统计，但你可以：

**方法一：使用 Google Analytics**
1. 注册 Google Analytics 账号
2. 创建媒体资源
3. 获取跟踪代码
4. 将代码添加到 index.html 的 `<head>` 中

**方法二：使用百度统计**
1. 访问 https://tongji.baidu.com/
2. 注册并添加网站
3. 获取统计代码
4. 添加到 index.html 中

**方法三：使用不蒜子**
1. 简单轻量的计数器
2. 无需注册，直接添加代码
3. 访问 https://busuanzi.ibruce.info/

### Q5: 分享到微信会被拦截吗？

**可能会被拦截的情况**：
- 短时间内大量分享
- 使用某些短链接服务
- 被用户举报

**避免被拦截的方法**：
- 不要频繁刷屏
- 使用 GitHub Pages 原始链接（比短链接更安全）
- 添加二维码作为备用方案
- 文案中不要包含诱导分享的内容

### Q6: 可以统计谁打开了链接吗？

GitHub Pages 默认不支持，但可以通过以下方法：

1. **使用短链接服务的统计功能**（部分服务提供）
2. **添加访问统计代码**（Google Analytics、百度统计）
3. **使用第三方服务**（如 Bitly、短链）

⚠️ **注意**：尊重用户隐私，不要过度收集数据。
